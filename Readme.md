Paptman2 — это свободно распространяемая обёртка для пакетного менеджера pacman (используемого в дистрибутивах Arch Linux и производных) с синтаксисом, похожим на пакетный менеджер apt (используемый в дистрибутивах Debian и производных), **с добавлением** AUR Helper (Arch User Repository Helper).

> Если вам нужен обычный Paptman без AUR Helper, используйте проект [Paptman](https://github.com/MarkGMX8X/paptman). 


---


## Примеры использования

| Команда | Действие |
|---------|----------|
| `paptman install <пакет>` | Установка из официальных репозиториев |
| `paptman install -r <пакет>` | Альтернатива (официальные репозитории) |
| `paptman install -p <пакет>` | Альтернатива 2 (официальные репозитории) |
| `paptman install -aur <пакет>` | Установка из AUR через yay |
| `paptman install -yay <пакет>` | Альтернатива (установка из AUR) |
| `paptman remove <пакет>` | Удаление пакета(ов) |
| `paptman update` | Обновление баз репозиториев |
| `paptman update -Sys` | Полное обновление системы (базы + пакеты) |
| `paptman search <пакет>` | Поиск пакета(ов) |

## Установка

Требуемые пакеты для установки - git base-devel
```bash
git clone https://github.com/MarkGMX8X/paptman2.git
cd paptman2
sudo cp paptman2 /usr/local/bin/
```
